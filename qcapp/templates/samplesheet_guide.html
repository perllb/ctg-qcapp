CTG SampleSheet guide

How to set up samplesheets for successful automation of transfer and analysis at CTG



Main steps

1. Automatic transfer: Name samplesheet in runfolder with prefix "CTG_SampleSheet."

2. Automatic pipeline execution: Describe which pipeline after prefix: "CTG_SampleSheet.name-of-pipeline.csv". See "Samplesheet name" in table below.

3. Every pipeline has specific requirements for samplesheet format and content. Click pipelines in table below for more details for the specific pipeline


General: Sample_ID and Sample_Name can ONLY contain "_" as a special character. All other special characters will cause demux to crash
